## 34.在排序数组中查找元素

[官方解析](<https://leetcode-cn.com/problems/minimum-size-subarray-sum/>)

### 1.难度:medium

这个问题挺重要的

### 2.重难点

* 数组是升序的
* 需考虑一下几种可能性

1.如果nums数组为空<br/>
2.如果target不在数组最小值到最大值范围内<br/>
3.target不在数组中

### 3.解法

* #### 解法一

解题思路:暴力解法<br/>

状态: 49.21%, 82.11%

```python
class Solution:
    def searchRange(self, nums: List[int], target: int) -> List[int]:
        length=len(nums)
        if length!=0:
            arr=[]
            for i in range(length):
                if target==nums[i]:
                    arr.append(i)
            len_arr=len(arr)
            return [min(arr),max(arr)] if len_arr!=0 else [-1,-1]
        else:
            return [-1,-1]
```
